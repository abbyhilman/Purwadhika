Git : Untracked, Modified & Staging

Untracked / Modified : Tempat untuk file yang baru saja dibuat atau mengalami perubahan.

Staging : Tempat untuk menaruh file apa saja yang hendak commit.

Commit : Tempat checkpoint dari kode kita.

git restore untuk mengembalikan file yang sudah di commit ke git

git revert untuk menghapus file tertentu yang sudah dicommit

git reset

Master branch adalah garis/cabang utama dari project yang sedang kita kerjakan

git rm --cached untuk mengahapus dari staging

test untuk fix ghost account
